angular.bind


描述：

上下文，函数以及参数动态绑定，返回值为绑定之后的函数. 其中args是可选的动态参数，self在fn中使用this调用。


使用方法：

angular.bind(self,fn,args );


参数：

参数名称 参数类型 描述
self Object fn的上下文对象，使用this调用
fn function 被绑定的function
args * 传入fn中的参数(可选的)

返回值：

返回动态绑定之后的函数
