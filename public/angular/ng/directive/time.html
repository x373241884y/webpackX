input[time]


描述：

输入时间验证和转换。在不支持HTML5的日期输入浏览器，文本元素将被使用。在这种情况下，文本必须输入有效的ISO-8601本地时间格式（HH：MM：SS），例如：14:57:00。模型必须是日期对象。这种结合会输出一个数据对象到1970年1月1日的模型，或本地日期新日期（1970，0，1，HH，MM，SS）。
该模型必须始终是日期对象，否则将抛出一个错误。无效的日期（日期对象的gettime()是NaN）将呈现为一个空字符串。 时区用于读/写日期的模型实例可以定义使用ngmodeloptions。默认情况下，这是浏览器的时区。


指令优先级

优先级别0。


属性：

参数名称 参数类型 描述
ngModel string 绑定的数据
name (optional) string 名称
ngRequired (optional) string 设置是否验证
required (optional) bool 验证不能为空
min (optional) number 设置值的最小值，如果超出这个最小值提示验证错误。
max (optional) number 设置值的最大值，如果超出这个最大值提示验证错误。
ngChange (optional) string 值改变时触发事件

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="timeExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>time例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('timeExample', [])
        .controller('DateController', ['$scope', function ($scope) {
        $scope.value = new Date(1970, 0, 1, 14, 57, 0);
        }]);
        </script>
</head>
<body>
        <form name="myForm" ng-controller="DateController as dateCtrl">
        Pick a between 8am and 5pm:
        <input type="time" id="exampleInput" name="input" ng-model="value"
        placeholder="HH:mm:ss" min="08:00:00" max="17:00:00" required />
        <span class="error" ng-show="myForm.input.$error.required">
            Required!
        </span>
        <span class="error" ng-show="myForm.input.$error.time">
            Not a valid date!
        </span>
        <tt>value = {{value | date: "HH:mm:ss"}}</tt><br />
        <tt>myForm.input.$valid = {{myForm.input.$valid}}</tt><br />
        <tt>myForm.input.$error = {{myForm.input.$error}}</tt><br />
        <tt>myForm.$valid = {{myForm.$valid}}</tt><br />
        <tt>myForm.$error.required = {{!!myForm.$error.required}}</tt><br />
        </form>
        </form>
</body>
</html>
