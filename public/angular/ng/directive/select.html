select


描述：

html select标签angular数据绑定。


ngOptions：

ngoptions属性可以使用数组或对象通过ngoptions表达式来动态生成一个列表元素的option 。
在许多情况下，ngrepeat可用于代替ngoptions选项元素达到类似的结果。然而，ngoptions提供了一些好处，如减少内存和不产生对每个重复的实例的一个新的scope，提高速度，以及提供怎样的<选择> 的模型更灵活分配通过选择的理解表达的一部分。ngoptions应该在select模型需要绑定到一个非字符串值。这是因为一个选项元素只能绑定到当前字符串值。
当select菜单选中其中一项，那么哪项的数组元素或是对象属性将绑定到模型的ngModel指令识别的变量中。
或者，一个单一的option元素，与一个空字符串的值，可以嵌套到select元素。这个元素将代表空或“未选择”选项。


select as：

select选中的值是绑定的ngModel的值但是这个值是as之前对应的值，用法：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="selectExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>selectas</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('selectExample', [])
        .controller('ExampleController', ['$scope', function ($scope) {
        $scope.users = [
        { name: 'a', id: '1' },
        { name: 'b', id: '2' },
        { name: 'c', id: '3' }
        ];
        $scope.selected = '2';//id的值，区分类型
        }]);
        </script>
</head>
<body ng-controller="ExampleController">
        <select ng-model="selected" ng-options="x.id as x.name for x in users"></select>
</body>
</html>

select as 和 track by：

不要在同一个表达式使用select as 和track by，track by和select as类似但track by 优先级更高点。用法：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="selectExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>track by例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('selectExample', [])
        .controller('ExampleController', ['$scope', function ($scope) {
        $scope.values = [{
        id: 1,
        label: 'aLabel',
        subItem: { name: 'aSubItem' }
        }, {
        id: 2,
        label: 'bLabel',
        subItem: { name: 'bSubItem' }
        }];
        $scope.selected = { id: 1 };
        $scope.selected1 = { name: 'bSubItem' };
        }]);
        </script>
</head>
<body ng-controller="ExampleController">
        <select ng-options="item.subItem as item.label for item in values track by item.id" ng-model="selected"></select>
        <select ng-options="item.subItem as item.label for item in values track by item.id" ng-model="selected1"></select>
</body>
</html>

指令优先级

优先级别0。


用法：

属性方式：
<select
        ng-model=""
        [name=""]
        [required=""]
        [ng-required=""]
        [ng-options=""]>
...
</select>

属性：

参数名称 参数类型 描述
ngModel string 绑定的数据
name (optional) string 名称
ngRequired (optional) string 设置是否验证
required (optional) bool 验证不能为空
ngOptions (optional) string 选项值表达式绑定
  • 数组数据源
    • label for value in array(就是循环数组数据);
    • select as label for value in array(select as用法);
    • label group by group for value in array(分组);
    • label group by group for value in array track by trackexpr(分组加track by用法);
    • label for value in array | orderBy:orderexpr track by trackexpr(排序加track by用法);
  • 对象数据源
    • label for (key , value) in object;
    • select as label for (key , value) in object
    • label group by group for (key, value) in object
    • select as label group by group for (key, value) in object

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="selectExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>select例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('selectExample', [])
        .controller('ExampleController', ['$scope', function ($scope) {
        $scope.colors = [
        { name: 'black', shade: 'dark' },
        { name: 'white', shade: 'light' },
        { name: 'red', shade: 'dark' },
        { name: 'blue', shade: 'dark' },
        { name: 'yellow', shade: 'light' }
        ];
        $scope.myColor = $scope.colors[2]; // red
        }]);
        </script>
</head>
<body>
        <div ng-controller="ExampleController">
        <ul>
        <li ng-repeat="color in colors">
                Name: <input ng-model="color.name">
                [<a href ng-click="colors.splice($index, 1)">X</a>]
        </li>
        <li>
                [<a href ng-click="colors.push({})">add</a>]
        </li>
        </ul>
        <hr />
        Color (null not allowed):
        <select ng-model="myColor" ng-options="color.name for color in colors"></select><br>
        Color (null allowed):
        <span class="nullable">
        <select ng-model="myColor" ng-options="color.name for color in colors">
        <option value="">-- choose color --</option>
        </select>
        </span><br />
        Color grouped by shade:
        <select ng-model="myColor" ng-options="color.name group by color.shade for color in colors"></select><br />
        Select <a href ng-click="myColor = { name:'not in list', shade: 'other' }">bogus</a>.<br>
        <hr />
        Currently selected: {{{selected_color:myColor}}}
        <div style="border:solid 1px black; height:20px"
        ng-style="{'background-color':myColor.name}">
        </div>

        </div>
</body>
</html>
