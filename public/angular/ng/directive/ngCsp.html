ngCsp


描述：

使CSP（内容安全策略）的支持,开发者模式，本人不才官方翻译理解不了，所以不放上来，等待大牛帮忙翻译。


指令优先级

优先级别0。


用法：

属性方式：
<!doctype html>
<html ng-app ng-csp>
...
...
</html>

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="cspExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngCsp例子</title>
        <script src="../angular-1.3.9/angular.js"></script>
        <script type="text/javascript">
        angular.module('cspExample', [])
        .controller('MainController', function () {
        this.counter = 0;
        this.inc = function () {
        this.counter++;
        };
        this.evil = function () {
        // jshint evil:true
        try {
        eval('1+sd');
        } catch (e) {
        this.evilError = e.message;
        }
        };
        });
        </script>
</head>
<body>
        <div ng-controller="MainController as ctrl">
        <div>
        <button ng-click="ctrl.inc()" id="inc">Increment</button>
        <span id="counter">
                {{ctrl.counter}}
        </span>
        </div>
        <div>
        <button ng-click="ctrl.evil()" id="evil">Evil</button>
        <span id="evilError">
                {{ctrl.evilError}}
        </span>
        </div>
        </div>
</body>
</html>
