ngClass


描述：

（官方翻译）ngclass指令允许你动态设置CSS类在一个HTML元素的数据绑定表达式，代表所有的类被添加。
该指令操作三种不同的方式，根据这三种类型的表达式计算结果为：
1、如果表达式计算为一个字符串，字符串应该是一个或多个空格分隔的类名称。
2、如果表达式计算为数组，数组中的每个元素应该是一个字符串，即一个或多个空格分隔的类名称。
3、如果表达式的计算结果为一个对象，然后用相应的密钥作为类名称还是值的对象的每个键值对。
4、如果特定类别已被设置，该指令将不会添加重复类。
当表达式更改时，将删除先前添加的类，然后添加新的类


指令优先级

优先级别0。


用法：

属性方式：
<input
        ng-class="">
...
</input>

类方式：
<ANY class="ng-class: ;"> ... </ANY>

属性：

参数名称 参数类型 描述
ngClass expression 添加的类

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngClass例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <style type="text/css">
        .strike {
        text-decoration: line-through;
        }
        .bold {
        font-weight: bold;
        }
        .red {
        color: red;
        }
        </style>
</head>
<body>
        <p ng-class="{strike: deleted, bold: important, red: error}">Map Syntax Example</p>
        <input type="checkbox" ng-model="deleted"> deleted (apply "strike" class)<br>
        <input type="checkbox" ng-model="important"> important (apply "bold" class)<br>
        <input type="checkbox" ng-model="error"> error (apply "red" class)
        <hr>
        <p ng-class="style">Using String Syntax</p>
        <input type="text" ng-model="style" placeholder="Type: bold strike red">
        <hr>
        <p ng-class="[style1, style2, style3]">Using Array Syntax</p>
        <input ng-model="style1" placeholder="Type: bold, strike or red"><br>
        <input ng-model="style2" placeholder="Type: bold, strike or red"><br>
        <input ng-model="style3" placeholder="Type: bold, strike or red"><br>
</body>
</html>
