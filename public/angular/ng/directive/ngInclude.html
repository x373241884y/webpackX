ngInclude


描述：

读取、编辑和包括外部HTML片段。
默认情况下，模板URL被限制在同一个域和应用程序文档中。这是通过调用$sce.getTrustedResourceUrl。你可以从其他信任值的域或协议加载模板。
此外，浏览器的同源策略和跨域资源共享（CORS）的政策可能会进一步限制模板是否被加载。例如，nginclude跨域请求所有的浏览器都无效。


指令优先级

次指令将创建一个新的scope。
优先级别400。


用法：

标签方式：
<ng-include
        src=""
        [onload=""]
        [autoscroll=""]>
...
</ng-include>

属性方式：
<ANY
        ng-include=""
        [onload=""]
        [autoscroll=""]>
...
</ANY>

类方式：
<ANY class="ng-include: ; [onload: ;] [autoscroll: ;]"> ... </ANY>

属性：

参数名称 参数类型 描述
ngInclude | src string 加载的地址
onload (optional) string 新的部分加载时触发一个表达式。
autoscroll (optional) string 设置滚动条。

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="includeExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngInclude例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('includeExample', [])
.controller('ExampleController', ['$scope', function ($scope) {
        $scope.templates =
        [{ name: 'template1.html', url: 'template1.html' },
        { name: 'template2.html', url: 'template2.html' }];
        $scope.template = $scope.templates[0];
}]);
        </script>
</head>
<body>
        <div ng-controller="ExampleController">
        <select ng-model="template" ng-options="t.name for t in templates">
        <option value="">(blank)</option>
        </select>
        url of the template: <tt>{{template.url}}</tt>
        <hr />
        <div class="slide-animate-container">
        <div class="slide-animate" ng-include="template.url"></div>
        </div>
        </div>
</body>
</html>
