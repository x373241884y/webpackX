ngRepeat


描述：

（官方翻译）指令的ngrepeat实例化一个模板集合中的每项。每个模板实例都有其自己的scope，其中给定的循环变量设置为当前集合项目，并将$index为项目索引。 就是循环。 特殊属性在每个模板实例的局部范围内暴露，包括：
名称 类型 描述
$index number 循环的索引值
$first boolean 是否第一次循环
$middle boolean 是否第一次和最后一次循环之间的循环。
$last boolean 是否最后一次循环。
$even boolean 是否循环的次数$index为偶数。
$odd boolen 是否循环的次数$index为奇数

可以用ngInit来创建这些属性全面，这个在嵌套ngrepeats时是很有用的。


循环的开始和结束位置：

循环的元素不只是一个dom元素，ngrepeat支持多个dom元素一起循环从ng-repeat-start起始位置开始到ng-repeat-end为结束位置。

<header ng-repeat-start="item in items">
  Header {{ item }}
</header>
<div class="body">
  Body {{ item }}
</div>
<footer ng-repeat-end>
  Footer {{ item }}
</footer>

里面循环的数组是["A","B"],结果如下：

<header>
  Header A
</header>
<div class="body">
  Body A
</div>
<footer>
  Footer A
</footer>
<header>
  Header B
</header>
<div class="body">
  Body B
</div>
<footer>
  Footer B
</footer>
对于ngrepeat自定义起始和结束点也支持所有其他HTML指令语法口味提供AngularJS

指令优先级

创建一个新的scope。 优先级别1000。


用法：

属性方式：
<ANY
        ng-repeat="">
...
</ANY>
