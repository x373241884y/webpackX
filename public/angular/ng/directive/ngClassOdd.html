ngClassOdd


描述：

（官方翻译）ngclassold和ngclasseven指令工作完全按照ngclass，除非他们与ngrepeat工作发生作用只在奇数（偶数）行。 这个指令只能应用在一个ngrepeat范围。


指令优先级

优先级别0。


用法：

属性方式：
<input
        ng-class-odd="">
...
</input>

类方式：
<ANY class="ng-class-odd: ;"> ... </ANY>

属性：

参数名称 参数类型 描述
ngClassOdd expression 添加的类

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8"/>
        <title>ngClassEven例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <style type="text/css">
        .odd {
        color: red;
        }
        .even {
        color: blue;
        }
        </style>
</head>
<body>
        <ol ng-init="names=['John', 'Mary', 'Cate', 'Suz']">
        <li ng-repeat="name in names">
        <span ng-class-odd="'odd'" ng-class-even="'even'">
                {{name}} &nbsp; &nbsp; &nbsp;
        </span>
        </li>
        </ol>
</body>
</html>
