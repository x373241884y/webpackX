ngModel


描述：

ngModel 指令将scope中的属性绑定在input、select、texttarea上，实现双向数据绑定。
ngmodel有效： 视图绑定到模型的其他指令，如输入、文本或选择要求。 提供验证行为（即要求，数量，电子邮件，网址）。 保持控制的状态（有效/无效，肮脏/原始，触摸/未触及，验证错误）。 设置相关的CSS类的元素（ng-valid, ng-invalid, ng-dirty, ng-pristine, ng-touched, ng-untouched）。 将控制与父窗体注册。


指令优先级

优先级别1。


css类：

  • ng-valid：model有效
  • ng-invalid:model无效
  • ng-valid-[key]: 通过$setValidity循环添加有效key。
  • ng-invalid-[key]: 通过$setValidity循环添加无效key。
  • ng-pristine: 控制没有被调用。
  • ng-dirty: 控制被调用。
  • ng-touched: 控制被去掉。
  • ng-untouched: 控制器没有被去掉。
  • ng-pending: $asyncValidators没有被兑现。

属性：

参数名称 参数类型 描述
ngList string 分隔符

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="inputExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngModel例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('inputExample', [])
        .controller('ExampleController', ['$scope', function ($scope) {
        $scope.val = '1';
        }]);
        </script>
        <style type="text/css">
        .my-input {
        -webkit-transition: all linear 0.5s;
        transition: all linear 0.5s;
        background: transparent;
        }
        .my-input.ng-invalid {
        color: white;
        background: red;
        }
        </style>
</head>
<body>
    Update input to see transitions when valid/invalid.
    Integer is a valid value.
        <form name="testForm" ng-controller="ExampleController">
        <input ng-model="val" ng-pattern="/^\d+$/" name="anim" class="my-input" />
        </form>
</body>
</html>
