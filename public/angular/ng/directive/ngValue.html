ngValue


描述：

对option或input[radio]绑定表达式的值，所以当选中元素，该元素的值就会赋值到绑定的ngmodel中。
ngValue支持option，但用ngValue只能赋值字符串给ngmodel，要用到对象用ngoptions。


指令优先级

优先级别0。


用法：

属性方式：
<input
        [ng-value=""]>
...
</input>

属性：

参数名称 参数类型 描述
ngValue (optional) string angular表达式

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="valueExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title></title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('valueExample', [])
        .controller('ExampleController', ['$scope', function ($scope) {
        $scope.names = ['pizza', 'unicorns', 'robots'];
        $scope.my = { favorite: 'unicorns' };
        }]);
        </script>
</head>
<body>
        <form ng-controller="ExampleController">
        <h2>Which is your favorite?</h2>
        <label ng-repeat="name in names" for="{{name}}">
            {{name}}
        <input type="radio"
        ng-model="my.favorite"
        ng-value="name"
        id="{{name}}"
        name="favorite">
        </label>
        <div>You chose {{my.favorite}}</div>
        </form>
</body>
</html>
