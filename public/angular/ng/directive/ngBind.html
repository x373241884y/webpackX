ngBind


描述：

设置ngbind属性，angular会将ngbind的值替换掉标签的内容，并更新文本内容的价值时，表达的变化。
通常情况下，不使用ngbind属性，而是使用双卷标记{ { } }。
如果网速比较慢浏览加载angularjs之前会显示{{ }}原始形态时，最好用ngbind来代替{{}}，因为ngbind是元素的属性，加载的时候看不到。


指令优先级

优先级别0。


用法：

属性方式：
<ANY
        ng-bind="">
...
</ANY>
类方式：
<ANY class="ng-bind: ;"> ... </ANY>

属性：

参数名称 参数类型 描述
ngBind expression 要替换的表达式

例子：

效果演示 
<script>
        angular.module('bindExample', [])
        .controller('ExampleController', ['$scope', function($scope) {
        $scope.name = 'Whirled';
        }]);
</script>
<div ng-controller="ExampleController">
  Enter name: <input type="text" ng-model="name"><br>
  Hello <span ng-bind="name"></span>!
</div>
