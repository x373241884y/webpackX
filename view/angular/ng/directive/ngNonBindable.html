ngNonBindable


描述：

ngnonbindable指令告诉angular不编译或结合当前的DOM元素的含量。这是有用的，如果该元素包含什么angular指令和绑定，但应该忽略angular。可能如果你有一个网站，显示的代码片段是这样的。


指令优先级

优先级别1000。


用法：

属性方式：
<ANY ng-non-bindable>
...
</ANY>

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8"/>
        <title>ngNonBindable例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
</head>
<body>
        <div>Normal: {{1 + 2}}</div>
        <div ng-non-bindable>Ignored: {{1 + 2}}</div>
</body>
</html>
