ngController


描述：

设置对应的controller控制器，熟悉mvc的人应该知道，对试图进行操作。
(官方翻译)ngController指令是一个试图控制类，这是一个关键在angular如何支持模型视图控制器设计模式原则方面。 angular 的MVC组件： 模型：模型是一个scope的属性；scope属性绑定在DOM元素中。 视图：模板（带数据绑定HTML）进入视图中被渲染。 控制器：ngController指定一个Controller类，这个类控制业务逻辑和模型的在视图的绑定。


指令优先级

这个指令将创建新的scope
优先级别500。


用法：

属性方式：
<ANY
        ng-controller="">
...
</ANY>

属性：

参数名称 参数类型 描述
ngController expression 指定声明的控制器类

这个例子是说明angularjs声明的controller是一个类，可以用js原型的方式来添加属性和方法，这种写法比较少用到，一般都是和scope一起控制视图，个人觉得angularjs的controller用于简单的控制和模型绑定，比较复杂的业务操作可以封装到angularjs的service或是factory当中

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="controllerAsExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngControllerl例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('controllerAsExample', [])
        .controller('SettingsController1', SettingsController1);
        function SettingsController1() {
        this.name = "John Smith";
        this.contacts = [
        { type: 'phone', value: '408 555 1212' },
        { type: 'email', value: 'john.smith@example.org' }];
        }
        SettingsController1.prototype.greet = function () {
        alert(this.name);
        };
        SettingsController1.prototype.addContact = function () {
        this.contacts.push({ type: 'email', value: 'yourname@example.org' });
        };
        SettingsController1.prototype.removeContact = function (contactToRemove) {
        var index = this.contacts.indexOf(contactToRemove);
        this.contacts.splice(index, 1);
        };
        SettingsController1.prototype.clearContact = function (contact) {
        contact.type = 'phone';
        contact.value = '';
        };
        </script>
</head>
<body>
        <div id="ctrl-as-exmpl" ng-controller="SettingsController1 as settings">
        Name: <input type="text" ng-model="settings.name" />
        [ <a href="" ng-click="settings.greet()">greet</a> ]<br />
        Contact:
        <ul>
        <li ng-repeat="contact in settings.contacts">
        <select ng-model="contact.type">
        <option>phone</option>
        <option>email</option>
        </select>
        <input type="text" ng-model="contact.value" />
                [ <a href="" ng-click="settings.clearContact(contact)">clear</a>
                | <a href="" ng-click="settings.removeContact(contact)">X</a> ]
        </li>
        <li>[ <a href="" ng-click="settings.addContact()">add</a> ]</li>
        </ul>
        </div>
</body>
</html>
