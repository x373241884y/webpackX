ngForm


描述：

表单名称。
（官方翻译）嵌套的表单指令别名。HTML表单元素不允许嵌套。它是有用的嵌套形式，例如，如果一个子组件需要确定有效。
注：ngform的目的是一组控制，但不是替换

标记的功能（如发布到服务器）


指令优先级

优先级别0。


用法：

标签方式：
<ng-form
        [name=""]>
...
</ng-form>

属性方式：
<ANY
        [ng-form=""]>
...
</ANY>

类方式：
<ANY class="[ng-form: ;]"> ... </ANY>

属性：

参数名称 参数类型 描述
ngForm | name string 表单名称。如果指定，则将将表单控制器发布到相关scope，根据此名称。
