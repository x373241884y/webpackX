ngList


描述：

在分隔的字符串和一个字符串数组之间转换的文本输入。默认的分隔符是一个逗号后面的空间相当于ng-list=", "。你可以指定一个自定义分隔符的nglist属性值例如，ng-list=" | "。
该指令的行为是由ngtrim属性的影响。 如果ngtrim设置为“false”然后空格在分离器和每个列表项的有效。这意味着，该指令的用户负责处理空白，但也允许你使用空格作为分隔符，如制表符或换行符。 否则，周围的空白分隔符时忽视了在每个列表项空白才加入模型之前。


指令优先级

优先级别0。


用法：

属性方式：
<input
        [ng-list=""]>
...
</input>

属性：

参数名称 参数类型 描述
ngList string 分隔符

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="listExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngList例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('listExample', [])
.controller('ExampleController', ['$scope', function ($scope) {
        $scope.names = ['morpheus', 'neo', 'trinity'];
}]);
        </script>
</head>
<body>
        <form name="myForm" ng-controller="ExampleController">
        List: <input name="namesInput" ng-model="names" ng-list required>
        <span class="error" ng-show="myForm.namesInput.$error.required">
            Required!
        </span>
        <br>
        <tt>names = {{names}}</tt><br />
        <tt>myForm.namesInput.$valid = {{myForm.namesInput.$valid}}</tt><br />
        <tt>myForm.namesInput.$error = {{myForm.namesInput.$error}}</tt><br />
        <tt>myForm.$valid = {{myForm.$valid}}</tt><br />
        <tt>myForm.$error.required = {{!!myForm.$error.required}}</tt><br />
        </form>
        <textarea ng-model="list" ng-list="&#10;" ng-trim="false"></textarea>
<pre>{{ list | json }}</pre>
</body>
</html>
