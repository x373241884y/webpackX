input[datetime-local]


描述：

随着时间的验证和转换输入。在不支持HTML5的日期输入浏览器，文本元素将被使用。在这种情况下，文本必须输入有效的ISO-8601本地DateTime格式（yyyy mm ddthh：MM：SS），例如：2010-12-28t14:57:00。
该模型必须始终是日期对象，否则将抛出一个错误。无效的日期（日期对象的gettime()是NaN）将呈现为一个空字符串。 时区用于读/写日期的模型实例可以定义使用ngmodeloptions。默认情况下，这是浏览器的时区。


指令优先级

优先级别0。


属性：

参数名称 参数类型 描述
ngModel string 绑定的数据
name (optional) string 名称
min string 设置最小的验证值，如果输入值小于mint提示错误，这必须是一个有效的ISO日期字符串（yyyy-mm-dd）。
max (optional) string 设置最大验证值，如果输入值大于最大提示错误。这必须是一个有效的ISO日期字符串（yyyy-mm-dd）。
required (optional) string 不为空验证
ngRequired (optional) bool 设置是否验证
ngChange (optional) string 值改变时触发事件

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="dateExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>datetime例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('dateExample', [])
        .controller('DateController', ['$scope', function ($scope) {
        $scope.value = new Date(2010, 11, 28, 14, 57);
        }]);
        </script>
</head>
<body>
        <form name="myForm" ng-controller="DateController as dateCtrl">
        Pick a date between in 2013:
        <input type="datetime-local" id="exampleInput" name="input" ng-model="value"
        placeholder="yyyy-MM-ddTHH:mm:ss" min="2001-01-01T00:00:00" max="2013-12-31T00:00:00" required />
        <span class="error" ng-show="myForm.input.$error.required">
            Required!
        </span>
        <span class="error" ng-show="myForm.input.$error.datetimelocal">
            Not a valid date!
        </span>
        <tt>value = {{value | date: "yyyy-MM-ddTHH:mm:ss"}}</tt><br />
        <tt>myForm.input.$valid = {{myForm.input.$valid}}</tt><br />
        <tt>myForm.input.$error = {{myForm.input.$error}}</tt><br />
        <tt>myForm.$valid = {{myForm.$valid}}</tt><br />
        <tt>myForm.$error.required = {{!!myForm.$error.required}}</tt><br />
        </form>
</body>
</html>
