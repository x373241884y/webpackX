ngBindHtml


描述：

插入表达式的html元素到自定的地方。


指令优先级

优先级别0。


用法：

属性方式：
<ANY
        ng-bind-html="">
...
</ANY>

属性：

参数名称 参数类型 描述
ngBindHtml expression 表达式

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="bindHtmlExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngBindHtml例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script src="../angular-1.3.0.14/angular-sanitize.js"></script>
        <script type="text/javascript">
        angular.module('bindHtmlExample', ['ngSanitize'])
        .controller('ExampleController', ['$scope', function ($scope) {
        $scope.myHTML =
        'I am an <code>HTML</code>string with ' +
        '<a href="#">links!</a> and other <em>stuff</em>';
        }]);
        </script>
</head>
<body>
        <div ng-controller="ExampleController">
        <p ng-bind-html="myHTML"></p>
        </div>
</body>
</html>
