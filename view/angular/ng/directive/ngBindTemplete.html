ngBindTemplate


描述：

设置ngBindTemplate属性，angular会将ngBindTemplate的值替换掉标签的内容。不同的ngbind，ngbindtemplate可以包含多个{ { } }表达式。这个指令是因为一些HTML元素（如标题和需要选择）不能包含span元素。


指令优先级

优先级别0。


用法：

属性方式：
<ANY
        ng-bind="">
...
</ANY>
类方式：
<ANY class="ng-bind: ;"> ... </ANY>

属性：

参数名称 参数类型 描述
ngBind expression 要替换的表达式

例子：

效果演示 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" ng-app="bindExample">
<head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>ngBindTemplate例子</title>
        <script src="../angular-1.3.0.14/angular.js"></script>
        <script type="text/javascript">
        angular.module('bindExample', [])
        .controller('ExampleController', ['$scope', function ($scope) {
        $scope.salutation = 'Hello';
        $scope.name = 'World';
        }]);
        </script>
</head>
<body>
        <div ng-controller="ExampleController">
        Salutation: <input type="text" ng-model="salutation"><br>
        Name: <input type="text" ng-model="name"><br>
        <pre ng-bind-template="{{salutation}} {{name}}!"></pre>
        </div>
</body>
</html>
